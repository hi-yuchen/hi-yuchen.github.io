# Yu Chen Academic Website

Source for the bilingual academic website at [hi-yuchen.github.io](https://hi-yuchen.github.io).

## Updating content

- Personal details and research interests: `_config.yml`
- Research publications and manuscripts: `_data/research.yml`
- Courses and teaching experience: `_data/teaching.yml`
- Home biography: `_pages/about.md`
- Public CV: `files/yu-chen-cv.pdf`

The site is built and deployed automatically by GitHub Pages after changes are merged into `master`.
